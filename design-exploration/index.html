After you pick

Reply with the variant letter (A / B / C), the headline you want, and any tweaks. Common tweaks:

  • · Different pink hue (warmer / cooler / more saturated)
  • · Swap one font from a different variant
  • · Different headline (mix & match the three options)
  • · More or less motion/decoration
  • · Cream vs pure white background
  • · Different feel for one specific section

Once locked, the chosen tokens (pink hex, font families, background) feed into Task 5 and Task 6 of the implementation plan, then build proceeds through Phases 3–9.

Logo recommendation

The current 500×500 line-art PNG works beautifully at footer/hero size, but at small header sizes the thin strokes go sub-pixel. Recommend: provide an SVG version (or a wordmark-only variant) so the header can scale cleanly. For now the variants pair the icon + a styled text wordmark.

Placeholder strategy

All product/photo placeholders are intentional gradients in each variant's pink. This is the launch-day default per spec — drop real photos into /public/gallery/<category>/ and the manifest auto-populates.
